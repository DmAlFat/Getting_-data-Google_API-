1) Для работы с данным проектом, необходимо клонировать его репозиторий с GitHub, открыть в используемой среде разработки и установить для него следующие зависимости:
sqlalchemy;
gspread;
pandas;
oauth2client.
2) Установить PostgreSQL и запомнить параметры соединения (База данных, Хост, Порт, Пользователь, Пароль).
3) Открыть в проекте файл connection_parameters.py и ввести в него соответствующие параметры из п.2 для соединения с БД.
4) Запустить скрипт main.py
5) Перейти в свою БД для контроля результата в таблице 'Getting_data'.
